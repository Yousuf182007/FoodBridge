How FoodBridge works

  1. Donors list surplus food with details (type, quantity, pickup time, and location/GPS).
  2. Nearby NGOs/volunteers get notified and claim the listing.
  3. Pickup and delivery are tracked for transparency and trust.

Stack for a production build

  • Frontend: React.js
  • Backend: Node.js + Express
  • Database: MongoDB / Firebase
  • APIs: Google Maps for location, Firebase Cloud Messaging for notifications

This demo is 100% static (no server). It saves data in your browser to make testing easy.
